
1. whisper
Библиотека для использования модели транскрипции от OpenAI. Она необходима для расшифровки речи в текст.

Установка:

bash
Копировать
Редактировать
pip install openai-whisper
2. sounddevice
Библиотека для записи аудио с микрофона. Она позволяет получать аудиофайл с микрофона в реальном времени.

Установка:

bash
Копировать
Редактировать
pip install sounddevice
3. soundfile
Библиотека для сохранения аудио в формате .wav (и других). Используется для записи и сохранения аудиофайлов.

Установка:

bash
Копировать
Редактировать
pip install soundfile
4. librosa
Библиотека для обработки и анализа аудио данных, в том числе для загрузки и подготовки звуковых файлов для обработки.

Установка:

bash
Копировать
Редактировать
pip install librosa
5. argparse (встроенная в Python)
Для обработки аргументов командной строки (например, выбора режима работы или длительности записи).

Устанавливать не нужно, так как она уже идет в стандартной библиотеке Python.

6. (опционально) numpy
Иногда для работы с массивами данных аудио полезно иметь numpy в системе.

Установка:

bash
Копировать
Редактировать
pip install numpy
